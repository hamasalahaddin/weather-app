# Weather App

A simple and responsive weather application built with JavaScript that fetches real-time weather data using the OpenWeatherMap API.

## Features
- Search weather by city
- Toggle between Celsius and Fahrenheit
- Responsive design (mobile & desktop)
- Dark / Light mode
- Loading spinner
- Error handling

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Use
